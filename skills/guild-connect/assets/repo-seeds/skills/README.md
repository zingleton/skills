# Skills

Fully-formatted agent skills you accumulate over time. Each skill is a folder
with a `SKILL.md` (a `name` + `description` frontmatter block) and any supporting
files it needs.

Your Chief of Staff discovers and uses these automatically. When you ask it to
build a new capability, it adds the skill here (after checking with you first).

When an informal instruction in `../Tools/` proves durable and worth packaging,
promote it into a full skill here.
