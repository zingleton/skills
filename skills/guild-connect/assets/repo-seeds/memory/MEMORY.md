# Memory Index

This is your portable memory — one fact per file, indexed here. It travels with
your `personal` repo across machines and AI assistants, so anything written here
follows you.

**Convention:** each memory is its own small markdown file in this folder, and
gets one line in the index below. Keep facts atomic — one idea per file — so they
are easy to update or remove individually.

Add entries as: `- [Short title](file-name.md) — one-line hook`
